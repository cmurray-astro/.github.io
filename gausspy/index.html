Motivation

What is GaussPy and why is it awesome?

Gaussian functions, described simply by three parameters and often physically well-motivated, provide a convenient basis set for a model. However, a set of Gaussian functions is not an orthogonal basis, so any solution composed of them is not unique. Furthermore, the crucial step of identifying the ``correct" number of Gaussians and their parameters is not straightforward. A standard approach is to make subjective, by-eye guesses. However, this approach is not scalable to increasingly large datasets, not to mention being inherently non-reproducible.

The goal of the GaussPy project is to solve this age-old problem! GaussPy implements the Autonomous Gaussian Decomposition algorithm, which leverages machine learning and derivative spectroscopy to identify all necessary parameters for initializing a multi-Gaussian fit (i.e., no eyeballs required!). For detailed information about the algorithm and code, please see our documentation.

On this page, you can explore motivating use cases for GaussPy, both simple and advanced, before diving into the code on our GitHub repository, which contains the code itself, and examples for how to use it.
